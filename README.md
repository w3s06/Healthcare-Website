# Healthcare-Website

A responsive, university‑developed healthcare information website built using **PHP, HTML, CSS, and JavaScript**. The project focuses on clean front‑end design, structured navigation, and dynamic content rendering through PHP. It demonstrates practical web development fundamentals, including page routing, form handling, layout structuring, and accessibility — all without using a SQL database backend.

---

## ✨ Features
- **Responsive and accessible UI** designed using modern HTML and CSS.
- **Dynamic page rendering** implemented with PHP to organise content and navigation.
- **Client-side interactivity** added using JavaScript for a smoother user experience.
- **Clear site structure** with multiple informative pages and consistent layout.
- **No database required** — the project runs fully on front-end and PHP logic.

## 🚀 Running the Project
1. Install a local PHP server (e.g., **XAMPP**, **WAMP**, or **MAMP**).  
2. Place the project folder inside your server’s `htdocs` or `www` directory.  
3. Start Apache.  
4. Visit:  
http://localhost/Healthcare-Website/src/enthub
